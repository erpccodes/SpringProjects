# File Upload and Management Web Application

This repository contains a simple file upload and management web application using Spring Boot framework. The application allows users to upload files and view the list of uploaded files.

## Prerequisites

Before running the application, make sure you have the following installed on your system:

1. Java Development Kit (JDK)
2. Apache Maven
3. Notepad (for editing and creating the files)

## How to Run the Application

1. Download and extract the project folder to your preferred location.


Running the Application
1.Open a Command Prompt or Terminal window.

2.Navigate to the project folder containing the files.

3.Build the project using Maven:
mvn clean install

4. Run the Spring Boot application:
mvn spring-boot:run

5.The application will start, and you can access it by opening your web browser and navigating to http://localhost:8080/.

6.You can now use the web application to upload files and view the list of uploaded files.
